Chapter 3 Assignment: Functions for Clarity

Read the chapter for how functions can be used to clarify a program. Use these tips and the other skills you have learned to draw one of the following in Processing:

  • snowman
  • bunny
  • propose an image for approval by your teacher

The code and resulting picture must include the following:

  • All shapes are used (line, ellipse or circle, rectangle or square)
  • The image can be easily identified (you can tell what the picture is)
  • At least one function with parameters is used
  • All functions are called properly
  • All function work as intended

Additionally, the code must be properly documented with sufficient comments and docstrings for all functions.
